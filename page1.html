HOW IT WORKS
① Text is converted to UTF-8 bytes → hex string
② Hex is split into 6-digit blocks → #RRGGBB colour codes
③ Remaining space padded with #000000 (black)
④ Each code becomes one pixel in a square 1:1 image
⑤ Decode reverses: pixels → hex → bytes → text
